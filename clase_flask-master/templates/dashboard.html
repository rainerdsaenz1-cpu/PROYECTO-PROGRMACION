¿Qué puedes hacer?

Como Operador, tu función es ingresar transacciones de manera segura. No mostramos opciones que no te corresponden para mantener una interfaz clara.

Si necesitas modificar transacciones existentes, contacta a un administrador.

🔒 Tu Seguridad es Importante

Como Operador, tu rol se limita a ingresar transacciones. Las siguientes protecciones están activas:

  • Validación de datos: Todos tus datos se validan antes de guardarse
  • Protección XSS: Tus entradas se sanitizan automáticamente
  • Seguridad NoSQL: No puedes ejecutar consultas especiales
  • Restricción de rol: Solo ves las funciones permitidas para tu perfil
